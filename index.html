    Projects

    Bugu Sport

    Responsibilities: Project leader, project management, full-stack development, operation maintenance, market research, promotion, and etc.

    Brief: Bugu Sport is a WeChat Mini Program on-campus sports whose user groups mainly focus on college and primary and secondary school students. Some functions include inviting to the game, joining the streetball, managing teams, organizing competitions, personal authentication, etc. Now there are 1400+ scanning visitors and 600+ registered users.
    Award: Won National-Level Second Prize & South China-Level First Prize by China Collegiate Computing Contest and third Prize in "Byte Dance" Cup Software Design Competition
    Copyright: The source code has been registered in Copyright Protection Center of China(CPCC).

    Orange Browser

    Responsibilities: Code developer (mainly focused modules: bookmark, history, quick page, splash page, UI layout), requirements research and analysis, interaction Design, technical Design.

    Brief: Orange Browser is an android application which developed during a six-week training program jointly held by ByteDance and South China university of Technology(SCUT).

    • See project

    ACIMG

    Responsibilities: Front end leader

    Brief: ACIMG(Arts & Culture Images) aims to create a nationwide image dataset of cultural relics and ancient buildings by college students from all over the country.

    • See project

    LTDRS

    Responsibilities: Front end leader, the connection of front and back end, the deployment of BigchainDB in linux environment, requirements analysis.

    Brief: LTDRS(Lecture Ticket Distribution&Record System) based on BigchainDB, which takes the advantages of BigchainDB distributed database and traditional blockchain, so that every lecture ticket issued in this system and its corresponding person can be recorded, which ensures the efficiency and fairness of lecture ticket distribution process in schools' activities.

    • See project

    Campus Life

    Responsibilities: Game designer and code developer(mainly focused modules: Game plots)

    Brief: Campus-Life is a game of education simulation of the C++ programming language, and its object is to help players learn more about or reminisce university life. It is a dialog-based MFC application, developed in the environment of Microsoft Visual Studio 2010, using static library MFC and several dynamic link libraries, which finished in two months.
    Award: Won the third prize in "Kingdom" cup Software Design Competition held by School of Software Engineering in SCUT.
